### 2026.04.22. Chapter 04-1. 로지스틱 회귀
* 로지스틱 회귀는 사실 선형 방정식을 사용하는 **분류 모델**
* 이진 분류
  - 하나의 선형 방정식 훈련 → 시그모이드 함수에 통과시켜 0~1 사이의 값을 만듦(=양성 클래스에 대한 확률)
  - 음성 클래스에 대한 확률 = 1- 양성 클래스에 대한 확률
* 다중 분류
  - 클래스 개수만큼 방정식을 훈련 → 소프트맥스 함수에 통과시켜 전체 클래스의 합이 항상 1이 되도록 만듦 
  - 각 클래스에 대한 확률 계산
* 시그모이드와 소프트맥스는 나중에 신경망 단계에서 또 나옴!

### 2026.04.20. Chapter 03-3.특성공학과 규제
**특성공학(늘리기)  →  규제 적용(릿지/라쏘)  →  강도 조절(alpha)  →  최적 모델 완성!**
* 데이터 특성 늘리기(PolynomialFeatures) $\rightarrow$ 특성을 무작정 늘리면 모델이 복잡해져 **과대적합** 발생
* 이를 해결하기 위 계수($w$)를 제어하는 **규제 적용** $\rightarrow$ 릿지/라쏘 회귀
  - 릿지 : 계수($w$)를 작게 만들어 모델을 부드럽게, 모든 특성을 다 안고감
  - 라쏘 : 중요하지 않은 계수($w$)를 0으로 만들어 쓸모없는 특성을 가차 없이 버림
* 하이퍼파라미터($\alpha$)를 설정하여 규제 강도를 조절
